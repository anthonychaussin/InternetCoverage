## Commandes mongo
### Create database :
```
use IOP
```
### Create table from json file :
```
mongoimport --jsonArray --db IOP --collection IOU --file IOU.json
```
### Create table from the other json file :
```
mongoimport --jsonArray --db IOP --collection CodeInfo --file CodeInfo.json
```
